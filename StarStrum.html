Star Strum

Mobile Application Prototype

Project Overview

This project is done in partial fulfillment of the requirements for the module CSD3515.

Star Strum is a music streaming application that allows you to explore new music, artists, and create custom playlists. The prototype allows you to sign in, view the current song that is playing, and return back to the home page.

Requirements

  • Create a single-screen landing page for a mobile app in Figma following either a skeuomorphic, flat, or neumorphic design philosophy.
  • Add a second screen and use the prototyping feature to link the screens together in Axure.
  • Transitions between screens must be triggered by realistic user interaction.

THEME

Design Style

For this assignment, we are required to create the design using skeumorphic, flat, or neumorphic style. I have chosen the neumorphic design style as that was something new to me.

Color Scheme

In general, I felt that neumorphic design style looks a little futuristic. Hence, I wanted the background to be of a galactic color scheme. I have also chosen to use pastel tones to make the design look softer.

LANDING PAGE

UI Components

Input Fields

To differentiate the input fields from the background, inner shadows are added to the input fields to create a recessed look and to make it look like an indentation on the surface. This allows the user to instinctively understand where to click and fill the text.

Buttons

To make the buttons distinct and have a protruding look, drop shadows are added. A white inner shadow was also added to the top left corner of the checkbox and sign-in button to create a reflective effect on the surface.

Background

Speaker

Since it is a music streaming app, I thought of creating a speaker to fill the background. Additionally, as the background of the page is already colorful and bright, I decided to make the speaker white to keep it simple and create a contrast from the background.

Waves

To fill the bottom part of the screen, I created a vector with a wavy pattern to sort of simulate sound waves. Keeping with the theme, I added inner shadows to give the waves depth.

HOME PAGE

Playlists

The home page is divided into several sections, such as recommended playlists based on genres and artists that the user listens to, as well as custom playlists created by the user themselves.

Search Bar

Users can also search for specific artists, albums, or songs using the search bar at the top of the screen.

Music Player

The music player is shown on the bottom of the screen to show users the song that is currently playing.

MUSIC PLAYER PAGE

Album

The layer behind the album photo is translucent to make it look like a glass frame.

Progress Bar

Inner shadows are added to the main progress bar to make it look indented, so that it will look like it is being filled up as the song progresses.

Standard Buttons

White drop shadows are added to the play, pause, skip forward and skip back button to make it look like there is light shining from behind the buttons.
